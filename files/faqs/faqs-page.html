Access the server from https://rstudio.ppls.ed.ac.uk


If you forgot your password and want to change it

1. Log in to EASE https://www.ease.ed.ac.uk/ using your university UUN and password.

2. set your RStudio password here https://rstudio.ppls.ed.ac.uk/setpass.php, the username will be the same as your UUN (make sure you type your UUN correctly).

3. Access the server from https://rstudio.ppls.ed.ac.uk


If you are not registered for the service

1. Log in to EASE https://www.ease.ed.ac.uk/ using your university UUN and password.

2. set your RStudio password here https://rstudio.ppls.ed.ac.uk/setpass.php, the username will be the same as your UUN (make sure you type your UUN correctly).

2A. If (2) didn't work, please write an email to ppls.psych.stats@ed.ac.uk specifying (a) that you cannot access the RStudio Server online and (b) your UUN.

3. Access the server from https://rstudio.ppls.ed.ac.uk
